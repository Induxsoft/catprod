# Prodserv(Catálogo de productos y servicios) .
El paquete Prodserv (Catálogo de productos y servicios) de V12 es el módulo que centraliza la administración de artículos, servicios y ensambles (kits), además de gestionar precios y organizar el catálogo de líneas de productos. Su objetivo es ofrecer una estructura clara y flexible para que las empresas puedan controlar su oferta comercial de manera eficiente y coherente con otros módulos del ERP.
